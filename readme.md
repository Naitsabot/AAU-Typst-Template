# AAU Typst Template (Naitsa)
AAU report template written in typst

## Quick-start guide for online usage

Manually copy the following into the Typst Online App:
- `single_file_template.typ` *and rename it `template.typ`* (for ease)
- `main.typ` (so you dont have to fiddle)
- `sources.bib` to hold your citations
- `assets/` alll content of the assets folder into a folder of the same name `assets`. Contains:
  - appendix.typ (for appendencies)
  - `AAU` Folder full of AAU assets used in the template
